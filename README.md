# YouTube Views Prediction (US)

Проект направлен на прогнозирование просмотров видео на YouTube в США. Используются данные о видео: количество лайков, дизлайков, комментариев, длина заголовка и описания, количество тегов, категория видео, время публикации и день недели.  

Выполнены следующие шаги:  
- Исследовательский анализ данных (EDA) с визуализациями распределений, выбросов и корреляций между признаками.  
- Предобработка данных: заполнение пропусков, кодирование категориальных признаков, масштабирование числовых признаков.  
- Создание признаков: длина заголовка и описания, количество тегов, час и день публикации.  
- Обучение и сравнение моделей Linear Regression, Random Forest и Gradient Boosting.  
- Оценка качества моделей по метрикам MAE, R² и RMSE.  
- Сохранение лучшей модели и предсказаний для тестовой выборки.  

Цель проекта — определить, какие факторы влияют на популярность видео и предсказывать количество просмотров на основе доступных признаков.
